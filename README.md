# 🌍 Travel Blog

A full-featured travel blog application built with React and TypeScript.
Users can create posts, upload images, and leave comments.

The project demonstrates scalable frontend architecture, typed state management, routing, form validation, and API integration.

## 🏗 Project Architecture

The project follows a feature-based structure:

```
src/
 ├── app/            # store configuration
 ├── features/       # business logic (posts, auth, comments)
 ├── components/     # reusable UI components
 ├── pages/          # route-level pages
 ├── hooks/          # custom hooks
 ├── utils/          # helpers
 └── styles/         # global styles
```

Key architectural decisions:

* Separation of UI and business logic
* Typed Redux store
* Centralized API layer
* Reusable form components
* Scalable folder structure
